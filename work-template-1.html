Workspaces

In FME Desktop, users can create workspaces which handles data transformations. They can then upload these workspaces to FME Server in order to automate them and trigger other events.

Workspaces as the Functional Unit

Previous versions of FME Server were focused on having actions as the functional unit of the application. In the redesign we are working on, we are gradually shifting to workspaces as the functional unit.

Do the Thing

By creating a central hub for workspaces we made the actions available from there. This is a dramatic shift from navigating to an action page and then finding the workspace you wanted. Find the workspace, do the thing.

Living Style Guide

I am spearheading the creation of a living style guide. We recognized the need for consistency across all user workflows, and that starts with the layout and individual elements of the page.

The living style guide uses the style sheets and javascript directly from our application so anytime an update is made, the styleguide updates automatically.
